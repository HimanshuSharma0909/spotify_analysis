# 🎵 Spotify Top 50 World Dashboard

## 📌 Project Overview

The Spotify Top 50 World Dashboard is a Power BI Business Intelligence project designed to analyze global music chart performance using Spotify Top 50 data.

The dashboard provides insights into:

* Song popularity
* Artist performance
* Explicit vs Non-Explicit content
* Album release strategies
* Chart rankings
* Music trends over time
* 
---

## 🎯 Business Problem

Spotify stakeholders need an interactive analytics solution to monitor chart performance and understand listener behavior across songs, artists, and album releases.

This dashboard helps answer questions such as:

* Which songs are the most popular?
* Which artists dominate global charts?
* Are explicit songs more popular?
* How do album types impact chart performance?
* What trends emerge over time?
